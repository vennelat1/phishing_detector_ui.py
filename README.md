# Advanced Phishing URL Detection System

## 📌 Description
This project is a machine learning-based cybersecurity tool that detects whether a URL is safe or phishing.

## 🚀 Features
- URL input with GUI
- Feature-based detection (length, symbols, keywords)
- Logistic Regression model
- Real-time prediction

## 🛠️ Technologies Used
- Python
- Scikit-learn
- Tkinter

## ▶️ How to Run
```bash
python phishing_detector_ui.py
